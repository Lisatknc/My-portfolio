    • FIT IT（レイアウト作成アプリ）

    概要 【解決したい悩み】
    引越しする際、自分の荷物が新居に入るかわからないストレスを軽減したい

    【FIT ITで叶うこと】
    ・部屋のレイアウト画像をアップするだけで、家具や家電の配置シュミレーションができる。
    ・サイズ入力が可能だから、自分の荷物が引っ越し先にフィットするか確認できる。
    ・事前に必要・不要な荷物がわかり、引っ越し時のストレスを軽減。
    ・IKEAのHPから家具の情報を取得して配置できるので、IKEAで家具を購入するべきか判断できる。
    制作期間 2週間
    開発人数 4人
    使用言語 Ruby / Javascript / HTML / CSS
